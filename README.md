# 📄 AI Invoice Generator / Multi-Language Invoice Extractor

## 🚀 Description
An AI-powered invoice extraction system that converts invoice images into structured data using a multimodal LLM. Users can upload invoices, and the system automatically extracts key details such as invoice number, client name, date, total amount, and line items. It supports multi-language invoices and delivers real-time results through an interactive web interface.

---

## 🧠 Technology
- **Language:** Python  
- **Framework:** Streamlit  
- **AI Model:** Gemini 2.5 Flash (Vision)  
- **Libraries:** google-generativeai, Pillow, python-dotenv, LangChain, PyPDF2, ChromaDB  

---

## ⚙️ Workflow
User Upload → Image Processing → Gemini Vision Model → Data Extraction → Structured JSON Output → UI Display  

---

## 📊 Result
- ~90%+ extraction accuracy  
- Supports multi-language invoices  
- Real-time processing with structured output  
- Automated extraction of invoice fields (number, date, total, items, etc.)
